# 参数查询

## 能力
在图纸参数索引中按需查询技术参数。

## 查询方式

### 1. 精确匹配
通过参数名或别名精确查询，如"总长度"→ 527.0mm

### 2. 语义别名
通过同义词映射匹配，如"孔径"="网孔"="目"="mesh"

### 3. 模糊分词
jieba 中文分词后按 TF-IDF 权重匹配关键词

### 4. 数值搜索
给定数值查找对应参数，如"50mm 是什么"

### 5. 极值查询
- max/min 尺寸
- 最大/最小圆角半径

### 6. 计数查询
"50mm 有几处" → 统计出现次数

## 参数类型

### 尺寸参数（有数值）
- 长度、宽度、高度、直径、半径、角度、厚度、间距
- 孔径、外径、内径

### 信息参数（无数值）
- 材质、色调、成形方法、技术要求、编织方式
- 防菌型、退火处理、滤网支数、使用温度
- 耐油性、耐酸性、耐寒性、耐热性

### 标题栏信息
- 图纸名称、图号、比例、分类、图幅
- 版本号、重要等级、标准、管理编号
- 材料、材料规格

### 签字栏信息
- 设计、审核、工艺、标准化、批准、归档
- 每个角色对应签名和日期

### BOM 表
- 部品编号、名称、材料、规格
