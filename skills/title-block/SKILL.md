## 签字栏提取内容
| 角色 | 说明 | 位置特征 |
|------|------|----------|
| 设计 | 设计者签名 | 右下角，x≈960-1010(侧板)/x≈385-435(印刷版) |
| 审核 | 审核者签名 | 通常在"设计"下方 |
| 工艺 | 工艺审核者 | 通常在"审核"下方 |
| 标准化 | 标准化审核者 | 通常在"工艺"下方 |
| 批准 | 批准者签名 | 通常在"标准化"下方 |
| 归档/再归档 | 归档日期 | 可能有重复归档 |

## 提取策略

### 标题栏
1. 识别 HHLayerShuxinglan 图层的文字
2. 标签-值配对：标签在上方行(y≈27)，值在下方行(y≈20)或同行右侧
3. x 坐标接近 + y 坐标相同/下方 → 配对
4. 大号文字(h≥5.0)优先作为图纸名称

### 签字栏
1. 限定签字栏区域坐标范围
2. 角色标签定位 + 签名/日期文本关联
3. 按 y 坐标分组形成表格行
4. 按 x 坐标排序形成列
