# BOM 表查询

## 能力
从图纸 BOM 表中提取结构化物料信息。

## 提取内容
| 字段 | 说明 |
|------|------|
| 部品编号 | 零件在 BOM 表中的编号 |
| 部品名称 | 零件描述名称 |
| 材料牌号 | 如 PS-B, PP, SPCC, SECC, SUS304 |
| 材料规格 | 如 7T, 1.2mm |
| 数量 | 用量 |

## 提取策略

### 1. 图层识别
- BOM 表文字通常位于 **HHLayerMingxilan** 图层
- 标题栏信息（图号/比例等）位于 **HHLayerShuxinglan** 图层

### 2. 行分组
- 按 y 坐标取整分组（round(y/5)*5）
- 同一 y 组 → 同一 BOM 行

### 3. 列排序
- 同一行内按 x 坐标排序
- 用 " | " 连接各列文本

### 4. 材料识别
- SPCC → 冷轧钢板
- SECC → 电镀锌钢板
- SUS304 → 不锈钢
- PP → 聚丙烯
- PS/PS-B → 聚苯乙烯
- BC 瓦楞纸 + 7T → 纸板包装
