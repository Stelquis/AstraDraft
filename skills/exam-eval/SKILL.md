# 考核评估

## 能力
对照标准答案 CSV，评估 Agent 回答的准确率。

## 输入
- 标准答案文件（CSV 格式）
- CAD 图纸文件（DWG/DXF）

## 输出
- 逐题正误表（期望 vs 实际）
- 总体准确率
- 分类统计：
  - 尺寸参数准确率
  - 信息参数准确率
  - 标题栏/签字栏准确率
  - BOM 表准确率

## 评估规则

### 数值类答案
- 精确匹配：期望值与实际值完全相等
- 容差匹配：|期望 - 实际| < 0.5% 视为正确
- 单位一致性：检查 mm/cm/m 单位转换

### 文本类答案
- 完全匹配：字符串完全相等
- 包含匹配：实际答案包含期望答案的子串
- 语义匹配：核心关键词一致（如"PS"和"聚苯乙烯"视为等效）

### 格式
```csv
问题编号,问题,期望答案,实际答案,是否正确,备注
Q01,总长度是多少,527mm,527.0 mm,True,
Q02,图号是什么,HX-001,未找到,False,LLM 兜底未启用
```

## 准确率计算
```
总体准确率 = 正确题数 / 总题数 × 100%
分类准确率 = 该类别正确题数 / 该类别总题数 × 100%
```

## 评估报告格式
```markdown
# 考核评估报告
- 图纸: filter_modify.dwg
- 总题数: 41
- 正确: 38
- 准确率: 92.7%

## 分类统计
| 分类 | 正确/总数 | 准确率 |
|------|----------|--------|
| 尺寸参数 | 18/19 | 94.7% |
| 信息参数 | 12/13 | 92.3% |
| 标题栏 | 5/5 | 100% |
| 签字栏 | 3/4 | 75% |

## 错题详情
| 编号 | 问题 | 期望 | 实际 | 原因 |
|------|------|------|------|------|
| ...
```
